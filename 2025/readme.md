## Workflow

1. **Generate aggregated data**: Run `party_press/analysis/climate_health_nexus.ipynb`
   → writes `data/indicator_5_3_3.csv` and `data/indicator_5_3_3_country.csv`

2. **Generate figures**: Run `code/create_figures.ipynb`
   → writes PDFs to `appendix/`

## Key changes from 2024

- Year range extended to include 2025 data (`year < 2026`)
- Classification model: `z-dickson/xlm-roberta-large-political-issues` (retrained on ~136k examples including public CAP data and GPT-5-mini zero-shot labels)
